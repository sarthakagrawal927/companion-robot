# Companion Robot

A "phone on wheels" home companion robot — an iPhone is the head (camera, mic, speaker, face-screen), a wheeled ESP32 base is the body, and a Mac running [Pace](https://github.com/HeyPace/pace) is the off-board brain it reaches over Wi-Fi.
